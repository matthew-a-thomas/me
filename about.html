Leaving comments

At the bottom of each article is a link to a GitHub issue holding comments for that article. There you’ll be able to use GitHub Flavored Markdown to style your comments.

If you want to participate in the conversation but don’t have a GitHub account then you’ll need to create one. Sorry for that extra step, but it saves me a lot of effort in setting up a secure comments system and controlling spam and bots.
